### The course covers the following topics:

- Introduction to statistical learning

-  Linear models: simple and multiple linear regression

- Classification methods: logistic regression and Support Vector Machines (SVM)

- Model evaluation and validation: cross-validation and performance assessment

- Tree-based learning methods: decision trees

- Ensemble learning methods: bagging and boosting

- Introduction to deep learning, with a focus on convolutional neural networks (CNN)

Each theoretical topic is complemented by a practical session (TP) in which students implement the methods using Python and apply them to real datasets.

### Evaluation
The final grade is based on:

- Participation (attendance and in-class questions) (20%)

- Grades from practical sessions (TP) (30%)

- A final written exam (50%)
